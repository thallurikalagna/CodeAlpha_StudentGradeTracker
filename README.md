# Student Grade Tracker System

## Description

Student Grade Tracker System is a Java console-based application developed using Object-Oriented Programming concepts and ArrayList. The system helps faculty manage student records and allows students to view their results securely.

## Features

* Faculty Login
* Student Login
* Add Student
* Search Student
* Update Student Marks
* Delete Student
* View All Students
* View Topper
* View Class Statistics
* View Pass Students
* View Fail Students

## Concepts Used

* Java
* Object-Oriented Programming (OOP)
* Classes and Objects
* ArrayList
* Methods
* Loops
* Conditional Statements

## Calculations Performed

* Average Marks
* Highest Subject Score
* Lowest Subject Score
* Grade Calculation
* Pass/Fail Status
* Class Statistics

## Files

* Main.java
* Student.java
* StudentManager.java
* TeacherPortal.java
* StudentPortal.java

## How to Run

Compile:

javac *.java

Run:

java Main
